# 05_CSS

## Nome: Gustavo Henrique Oliveira Pestana

## Descrição da atividade

Atividade de Web I.

Para essa atividade, eu criei um site utilizando elementos de HTML, como textos, imagens, vídeo do YouTube, listas e sublistas
e links externos, além de elementos de estilização utilizando o CSS, como cores, definição de margens, conteúdos definidos com o flexbox, estilos de fontes, entre outros.

O site foi desenvolvido com o tema de jogos retrô, onde as cores, fontes e conteúdos foram criados a fim de corresponder com o tema selecionado. A ideia surgiu pelo gosto pessoal com relação a esse estilo de artes presentes em jogos antigos e atuais. Os conteúdos apresentados no site como os jogos, o console e as informações relacionadas aos mesmos são fictícios.
